# Mi Sitio Web Gamer

Este es un proyecto de página web personal estilo gamer, diseñado para ser publicado en GitHub Pages.

![Preview](https://upload.wikimedia.org/wikipedia/commons/0/0d/PS5_console.png)

## Descripción

La web incluye secciones de:

* **Inicio**: Introducción y bienvenida.
* **Proyectos**: Lista de proyectos destacados.
* **Contacto**: Información para contacto y redes sociales.
* **Consolas**: Información y guías para PlayStation, Xbox y Nintendo.
* **PC**: Builds, software y juegos de PC.

La estética es **gamer** con colores oscuros, glow neón, efectos hover en imágenes y un header tipo banner con fondo de videojuegos.

## Características

* Diseño responsivo adaptable a móviles y desktop.
* Menú con animación de glow al pasar el ratón.
* Tarjetas con imágenes y sombra, efecto hover.
* Estilo moderno y temático gamer.

## Tecnologías usadas

* HTML5
* CSS3 (estilos con variables, gradientes y efectos)

## Cómo usar

1. Clonar o descargar el repositorio.
2. Subir los archivos a GitHub.
3. Activar GitHub Pages desde la configuración del repositorio.
4. Abrir la URL generada por GitHub Pages.
